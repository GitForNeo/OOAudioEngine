# OOAudioEngine
A audio engine based on audio unit.simple to use.

#### Cocoapods

Add the following to your project's Podfile:
```ruby
pod 'oyh5233/OOAudioEngine', '~> 0.0.2'
```

#### Carthage

Add the following to your project's Cartfile:
```ruby
github "oyh5233/OOAudioEngine"
```
